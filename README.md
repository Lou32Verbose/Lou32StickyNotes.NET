# Lou32StickyNotes.NET
Lou32Sticky Notes .NET 

Lou32StickyNotes.NET is a .NET port of the Lou32StickyNotes Classic application (itself a clone of the original Sticky Notes application for Windows 7), with the goal of making it cross-platform.
It is written in C#, using Avalonia UI, and is based on the .NET 9.0 SDK.
